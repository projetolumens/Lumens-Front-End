Projeto Lumens

O Lumens foi criado para ser apresentado como Projeto Final de grupo da Turma-43 do Curso Bootcamp para Desenvolvedores Web Full Stack Java da ONG Generation Brasil. Com o objetivo de apresentar soluções tecnológicas para alcançar a ODS 7 - Energia Acessível e Limpa. Somos um e-Commerce focado em ajudar e conscientizar uma parte da população vulnerável economicamente, oferecendo informações e suporte para descartes e troca de produtos, de maneira ecológica, em conjunto com empresas e pontos ecológicos que visam o desenvolvimento sustentável.

Como funciona o descarte:

Separe os produtos eletrônicos que deseja descartar de acordo com a categoria. Não desmonte os produtos e, se o aparelho tiver bateria, não a remova. Você pode separar as pilhas comuns em uma sacola. Leve-os para o posto de coleta mais próximo ou agende a coleta na sua casa.

Como funciona o cashback:

Ao realizar o descarte do seu lixo eletrônico nos pontos de coleta de nossas empresas parceiras ou através do agendamento da coleta domiciliar, você receberá um código para ser convertido em cashback, que poderá ser usado para comprar os produtos em nosso site.

Desigualdade

Apesar de grande parte da população global ter acesso à eletricidade, a desigualdade ainda é uma grande problema para atingir o Objetivo de Desenvolvimento Sustentável que visa garantir energia acessível, confiável, sustentável e moderna para todos (ODS 7) na África Subsaariana por exemplo, a quantidade de famílias sem energia elétrica aumentou, pois nessa área praticamente não houve progresso, desde 2010, lembrando que o impacto da covid-19 tornou o serviço inacessível para cerca de 30 milhões de pessoas no mundo. Para preencher essas lacunas, o objetivo é a ampliação das energias renováveis.

Anterior Próximo

Eficiência Energética

A substituição de itens como lâmpada incandescente por lâmpadas de LED, promove economia de até 90% em relação às incandescentes, mas para que essa economia ocorra é importante que a população esteja informada, com esse objetivo foi criado o Programa Brasileiro de Etiquetagem (PBE) e existe o Programa Nacional de Conservação de Energia Elétrica, o Procel, que fornece informações sobre a eficiência energética dos equipamentos, assim na hora de trocar, podemos escolher produtos cada vez mais eficientes. O cliente então saberá que quando escolhe um produto com o selo, além da economia de energia elétrica, prejudicará menos o meio ambiente do que produtos sem selo.

Ferramentas Usadas

SpringBoot

É um framework desenvolvido para a plataforma Java baseado nos padrões de projetos (Design Patterns), inversão de controle e injeção de dependência.É constituído por diversos e completos módulos capazes de dar um boost na aplicação Java.

Java

Java é um tipo de linguagem de programação criada e comercializada pela Sun Microsystems desde 1995. É definida como uma linguagem orientada a objetos. Sua intenção é permitir que os desenvolvedores escrevam o programa apenas uma vez e o executem por meio de qualquer dispositivo.

Bootstrap

É um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.

Angular

Angular é uma plataforma e framework para construção da interface de aplicações usando HTML, CSS e, principalmente, JavaScript, criada pelos desenvolvedores da Google.
